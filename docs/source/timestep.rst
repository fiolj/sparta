################
timestep command
################

.. _timestep-syntax:

*******
Syntax:
*******

::

   timestep dt

dt = timestep size (time units)

.. _timestep-examples:

*********
Examples:
*********

::

   timestep 2.0 
   timestep 0.003

.. _timestep-descriptio:

************
Description:
************

Set the timestep size for subsequent simulations.
