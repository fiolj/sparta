.. _unfix-command:



#############
unfix command
#############




.. _unfix-syntax:



*******
Syntax:
*******





::



   unfix fix-ID




fix-ID = ID of a previously defined fix




.. _unfix-examples:



*********
Examples:
*********





::



   unfix 2
   unfix lower-boundary




.. _unfix-descriptio:



************
Description:
************




Delete a fix that was previously defined with a :ref:`fix<fix>`
command.
