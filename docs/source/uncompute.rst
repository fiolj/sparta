#################
uncompute command
#################

.. _uncompute-syntax:

*******
Syntax:
*******

::

   uncompute compute-ID

   - compute-ID = ID of a previously defined compute

.. _uncompute-examples:

*********
Examples:
*********

::

   uncompute 2
   uncompute lower-boundary

.. _uncompute-descriptio:

************
Description:
************

Delete a compute that was previously defined with a :ref:`compute<compute>`
command.
