##############
undump command
##############


**Syntax:**

::

   undump dump-ID 

-  dump-ID = ID of previously defined dump

**Examples:**

::

   undump mine
   undump 2 

**Description:**

Delete a dump that was previously defined with a :ref:`command-dump`. This also closes the file associated with the dump.

**Restrictions:** none

**Related commands:**

:ref:`command-dump`

**Default:** none
