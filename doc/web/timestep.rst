################
timestep command
################

**Syntax:**

::

   timestep dt 

-  dt = timestep size (time units)

**Examples:**

::

   timestep 2.0 
   timestep 0.003 

**Description:**

Set the timestep size for subsequent simulations.

**Restrictions:** none

**Related commands:**

:ref:`command-run`

**Default:**

::

   timestep 1.0
