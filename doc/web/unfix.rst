#############
unfix command
#############

**Syntax:**

::

   unfix fix-ID 

-  fix-ID = ID of a previously defined fix

**Examples:**

::

   unfix 2
   unfix lower-boundary 

**Description:**

Delete a fix that was previously defined with a `fix <fix.html>`__
command.

**Restrictions:** none

**Related commands:**

:ref:`command-fix`

**Default:** none
